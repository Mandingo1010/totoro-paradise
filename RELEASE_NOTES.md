# 🦫 Totoro Paradise v2.0.4 Release Notes

## 🎉 新版本发布！

这是 Totoro Paradise 的便携版发布，专为没有编程经验的用户设计，**下载即用，无需复杂配置**！

## 📥 下载说明

### Windows 用户 (推荐)
下载 `totoro-paradise-v2.0.4-windows.zip` (约 3.3MB)

### 系统要求
- Windows 10/11
- Node.js 16+ (程序会自动检测，如未安装会提示下载地址)

## 🚀 使用方法

1. **下载解压**: 下载zip文件并解压到任意文件夹
2. **一键启动**: 双击 `启动 Totoro Paradise.bat` 文件
3. **自动打开**: 等待几秒钟，浏览器会自动打开程序界面
4. **开始使用**: 在网页中选择功能并开始使用

## ✨ 主要功能

### 🏃‍♂️ 阳光跑 (Sun Run)
- 支持学校固定路线跑步模拟
- 自动生成真实的跑步轨迹数据
- 兼容多种学校系统

### 🆓 自由跑 (Free Run) ⭐ 核心功能
- **完全自定义**: 0.5-20公里距离，3-25km/h速度
- **预设模板**: 轻松跑/标准跑/挑战跑三种模式
- **批量执行**: 支持1-10次连续提交，智能间隔
- **真实数据**: 自动计算配速、卡路里、步数等
- **记录管理**: 查看历史、筛选导出

### 📊 记录管理
- 完整的跑步历史记录
- 按日期、类型、距离筛选
- 支持数据导出

## 🔧 常见问题

### Q: 提示"未检测到 Node.js"？
**A**: 访问 https://nodejs.org 下载安装 LTS 版本即可

### Q: 浏览器没有自动打开？
**A**: 手动打开浏览器访问 http://localhost:3000

### Q: 如何停止程序？
**A**: 关闭黑色命令行窗口即可
